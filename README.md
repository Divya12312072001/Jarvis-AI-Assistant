# Jarvis AI Assistant

A voice-controlled desktop assistant developed using Python and Tkinter.

## Features

* Voice Recognition
* Text-to-Speech Response
* Open Google
* Open YouTube
* Open Wikipedia
* Wikipedia Search
* Open Calculator
* Open Notepad
* Real-Time Clock
* Command History
* GUI Interface with Voice Animation

## Technologies Used

* Python
* Tkinter
* SpeechRecognition
* pyttsx3
* Wikipedia API
